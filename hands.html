Hands

Overview

This is a small game that can train the ability to think separately and with full attention. The screen will show the word in different colors, such as the word “Green” in red. It is a challenge to separate the meaning of words and the color of words.

Role

I had the initial concept of this game and the code part of coding as well as making the cards in a two-person team.

How to play?

Players need to find the card with the corresponding color, which is the word’s color and show it to the camera.

Player Flow

  1. Enter the game (click game start button)
  2. Take a headshot (press Esc to take a picture)
  3. Start playing the game (show timer and score on the right side and have a random picture to synthesize with the headshot)
  4. Time up and end the game
Tools:

Dev C++, OpenCV, Webcam
