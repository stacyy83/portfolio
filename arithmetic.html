Arithmetic Excercise

Overview

In order to comprehend a new coding language C# by myself, my final project advisor and professor asked me to undertake a small project by using this new coding language, and I think it is an efficient way to learn it. The young student in remote areas does not have abundant learning resources. In order to enhance their ability of calculation, I created this project to assist them in practicing calculation as a learning resource for them.

Designed for Whom?

The young student in remote areas.

Fuction Introduction

Before the game starts:
  1. Users can insert how many questions they want to practice with (within 10 questions) or they can skip the input box to set the random number of questions.
  2. Users can set the difficulty which has three degrees:
    Easy: two digit number arithmetic.
    Normal: three and four digit number arithmetic.
    Difficult: beyond five digit number arithmetic
  3. Users can select the math operators they want to practice; it provides a single operator or multi-operator. Moreover, they can choose not to carry or random carry.
Start:
  1. This practice has a timer to show how much time the user duly took during the practice. Besides, it also reveals the recommended finish time for users as a reference.
Result:
  1. The result will show the correct answer to users if they empty the input box or have a wrong answer. It also shows how many correct and incorrect answers were attained and the correct ratio of this practice.
