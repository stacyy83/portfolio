Emojis World

Link to the blog for more detail
P5.js demo

Overview

Emojis World is a project that replaces every pixel in the video with emojis. Additionally, it combines with machine learning to detect the specific pose and color.

Tools

P5.js, Machine Learning, Arduino

Interaction

  • - Heart Pose: Changes to heart emojis.
  • - Heart Pose: Changes to heart emojis.
  • - Detects the blue/red/green items: Changes to blue/red/green emojis.
